Syntax

Maximum board size is 19. 
var goboard = new GoBoard([container id], [length], [board size]);

19 x 19 board

Render a 19x19 Go board in a div with id "canvas" of length 500px:

var goboard = new GoBoard("canvas", 500, 19);

9x9 board

var goboard = new GoBoard("canvas", 300, 9);

13x13 board

var goboard = new GoBoard("canvas", 300, 13);
